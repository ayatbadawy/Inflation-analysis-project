📊 Inflation Data Analysis Project 📝 Overview This project is a Data Analysis of Inflation Rates using Power BI. The goal is to explore, analyze, and visualize inflation trends over time to understand how prices change and how it affects the economy. 🎯 Objectives Analyze inflation trends over different time periods Identify countries/regions with highest and lowest inflation Visualize changes in inflation clearly using interactive dashboards Extract meaningful insights to support decision making 🛠️ Tools Used Power BI Excel (for data cleaning and preparation) Data Visualization Techniques 📊 Dashboard Features Interactive charts and filters (Slicers) Trend analysis over time Comparison between regions/countries Key KPIs for inflation rates 🔍 Key Insights Inflation rates show noticeable variation across time periods Some regions experience consistently higher inflation External economic factors strongly influence inflation trends 📌 Files Included Power BI Dashboard (.pbix) Dataset (Excel file) README file 🚀 Conclusion This project helps in understanding inflation behavior and how data visualization can simplify complex economic data into clear insights.
